Pasal 001: Perilaku Dan Etika

  1. Hormati semua Pemain Dan Admin, Semua Player yang ada di server.
  2. Jangan membawa masalah pribadi kedalam server atau masalah server lain.
  3. Selesaikan masalah Secara Private Dan Baik tanpa melibatkan server Dan player lain.
  4. Tidak diperbolehkan melakukan pelecehan kepada member Dan player lain.
  5. Bahasa kasar Dan deskriminasi seperti Sara Dan lain sebagainya Dilarang keras di server.
  6. Dilarang keras melakukan Griefing Base player Dan bangunan server lain.
  7. Dilarang keras Mencuri barang member lain.
  8. Dilarang melakukan raiding antar clan atau base tanpa persetujuan admin.
  9. Duel PvP bisa dilakukan di arena saja.
  10. Player boleh melakukan taruhan kepada player lain di arena jika player yang melakukan duel menang player yg melakukan taruhan akan Mendapatkan 2x Dari taruhan mereka.
  11. Dilarang keras Hacking,Cheat Menggunakan X-ray,fly,exploit Dan lain sebagainya.
  12. Dilarang keras membuat Lag machine atau mesin besar di server.
  13. Max Portal Size untuk goldfarm 20,Max 2 Portal.

Pasal 002: Roleplay Rpg & Bangunan

  1. Ikuti Lore Dari server tersebut.
  2. Kalian boleh membangun dunia fantasi kalian sendiri disini.
  3. Pilih quest Dan role dengan bijak Dan kalian bisa membuat cerita kalian sendiri.
  4. Untuk pendaftaran clan Dan role bisa bilang ke admin.
  5. Setiap orang yg memilih role bisa tag admin untuk Mendapatkan rank role tersebut.
  6. Bijak lah dalam roleplay Dan rpg ini.
  7. Untuk pembangunan agar aman kalian bisa menggunakan claim block. (di How to play and mod)
  8. Hindari bangunan yg terlalu dekat agar tidak merusak estetika server.
  9. Player hanya bisa membangun base di area yang ditentukan admin Dan tidak boleh membangun area base lebih Dari 5k.
  10. World border untuk menaruh block 5k untuk resources kalian bisa keluar Dari world border tapi jangan membuat base disitu.

Pasal 003:PvP & PvE

  1. Pvp hanya di area tertentu dengan persetujuan staff.
  2. PvE Seperti Raid, Dungeon, boss fight,hunts.
  3. Untuk boss jika kalian berhasil Melawan boss tersebut silahkan japri ke admin untuk rank & rewards Tambahan nya.

Pasal 004: Ekonomi Dan Item

  1. Sistem ekonomi adil: User shop, auction house,bank,admin shop.
  2. Item langka Dan enchantment dapat melalui event Dan quest resmi.
  3. No dupe atau exploit untuk item2 langka.
  4. Player dapat membuat shop sendiri Dan menaruh chest shop dishop tersebut. Setidaknya buat bangunan seperti toko.
  5. Warp shop player jika ingin mendaftarkan shop mereka bisa chat admin Dan akan admin buatkan. (tidak pasti)
